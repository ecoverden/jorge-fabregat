# Jorge Fabregat — Landing

Landing page ultra-minimalista para Jorge Fabregat, asesor patrimonial independiente.

## Stack

HTML5 + CSS3 + JavaScript vanilla. Sin frameworks, sin build step.

## Estructura

```
landing-jorge-fabregat/
├── public/
│   ├── index.html
│   ├── style.css
│   └── main.js
├── vercel.json
├── package.json
└── README.md
```

## Preview local

```bash
npm start
# o directamente:
npx serve public -p 3000
```

Abrir `http://localhost:3000`.

## Deploy

Conectado a Vercel a través del repositorio de GitHub. Cada push a `main` despliega automáticamente.
